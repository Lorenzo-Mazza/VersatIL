# Research Scripts
Here we only keep scripts for research that are not part of the main codebase.
Note that these scripts may be removed without notice, once the library is officially released.
The purpose of these scripts is quick prototyping and experimentation with trained policies (e.g. additional insights, visualizations, etc.).
Ideally, the functionalities included in these scripts should eventually be integrated into the main codebase as clean and reusable modules and packages.

## Guidelines
Please do not use these scripts to test new features or functionalities of the main codebase.
Use development branches and pull requests for that purpose.
Please do not use these scripts to "test" functionalities of the main codebase. Use unit tests and integration tests instead.
Please do not include any additional dependencies in these scripts.
Please do not include broken or incomplete scripts.
If you have any questions regarding these scripts, please contact the authors directly.
